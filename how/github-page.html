How to crete your own github page

Steps:

  1. Create an account on Github
  2. Click on your account image (top rigth corner of the site)
  3. Click on "Your repositories"
  4. Click on " New"
  5. On "Repository name" field add: <your-user-name>.github.io
  6. Create the new repository
  7. In the repository page, on the Quick setup section, click on "creating new file"
  8. On "Name your file..." field add: index.html
  9. In the text editor, on line 1 write "<h1>My own github page</h1>"
  10. Click on Commit new file
  11. Navigate to https://<your-user-name>.github.io/
